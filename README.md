FUSE 01 で制作したゲーム
========================
* Bチーム
* enchant.jsを使用して開発

フォルダ構造
------------
* src     JavaScriptのソース
* fuse01b 実行可能な環境

ビルドなど
----------
* make allを実行すると、srcの中のファイルが連結されて、fuse01bの中にall.jsという名前で格納される。
* make releaseを実行すると、9leapに投稿するためのfuse01b.zipが生成される。
* make cleanを実行すると、fuse01b.zipが消去される。
